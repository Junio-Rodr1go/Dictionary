A

Atributo: Informação adicional em uma tag HTML que modifica seu comportamento. Exemplo: <a href="https://example.com">Link</a>.

Alt (atributo): Define um texto alternativo para imagens, útil para acessibilidade e SEO. Exemplo: <img src="imagem.jpg" alt="Descrição da imagem">.

B

Body: Elemento <body>, onde fica o conteúdo visível da página HTML.

Break Line (<br>): Tag para quebrar linha dentro de um texto.

C

CSS (Cascading Style Sheets): Linguagem usada para estilizar elementos HTML.

Class (atributo): Define uma ou mais classes para um elemento, útil para estilização com CSS e manipulação com JavaScript. Exemplo: <div class="caixa">.

D

DOCTYPE: Declaração que informa ao navegador a versão do HTML usada no documento.

Div: Elemento <div> usado para agrupar e estruturar conteúdo.

E

Elemento: Estrutura completa de uma tag HTML, incluindo abertura, conteúdo e fechamento, se aplicável.

Em (ênfase): Tag <em> para destacar um texto em itálico.

F

Formulário: Conjunto de elementos HTML (<form>, <input>, <button>) usados para entrada de dados do usuário.

H

Head: Seção <head> do HTML que contém metadados, links para CSS, scripts e título da página.

Header: Elemento <header> usado para cabeçalhos de páginas ou seções.

Hiperlink: Criado com a tag <a>, permite navegar para outras páginas.

I

ID (atributo): Identificador único para um elemento HTML.

Iframe: Elemento <iframe> usado para incorporar páginas dentro de outras páginas.

Imagem: Elemento <img> usado para exibir imagens.

L

Lista Ordenada (<ol>): Lista numerada.

Lista Não Ordenada (<ul>): Lista com marcadores.

M

Meta: Elemento <meta> usado para metadados, como codificação e SEO.

N

Nav: Elemento <nav> usado para menus de navegação.

P

Parágrafo (<p>): Tag para estruturar textos.

Placeholder: Atributo usado em <input> para exibir um texto de dica dentro do campo.

S

Section: Elemento <section> usado para dividir seções do conteúdo.

Span: Elemento <span> usado para estilizar partes específicas do texto.

T

Tabela: Criada com <table>, <tr> (linha), <td> (célula).

Title: Define o título da página (exibido na aba do navegador).

Tags: Marcadores HTML, como <h1>, <p>, <a>.

U

URL (Uniform Resource Locator): Endereço de um recurso na web.
